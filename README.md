# Smart Attendance System using Face Recognition

A production-grade Smart Attendance System built using Python, FastAPI, OpenCV, and face recognition.

## Features
- Real-time face recognition
- Automatic attendance marking
- No duplicate attendance per day
- Admin dashboard
- Export attendance reports

## Tech Stack
- Backend: FastAPI
- AI: OpenCV, face_recognition
- Database: SQLite (scalable to MySQL)
- Frontend: HTML, CSS, Bootstrap, JavaScript

## Use Cases
- Colleges & Schools
- Training institutes
- Hackathons & Startup demos
